# Hangman

A simple hangman game made using the React,Typescript, Vite. 

## Tech Stack

**Client:** React , Vite , TS
  
## Run Locally

Clone the project

```bash
  git clone https://github.com/poudyalamit/Hangman
```

Go to the project directory

```bash
  cd Hangman
```

Install dependencies

```bash
  npm install
```

```bash
  npm run dev
```
